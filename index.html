How it works

1

Expose Hermes

Enable the Hermes API server on your VPS and put HTTPS in front of it. The robot always dials out — no port-forwarding at home.

2

Install the app

Add it from the Reachy Mini dashboard, then enter your Hermes URL, API key and an OpenAI key on the settings page.

3

Just talk

Voice activity detection hears your sentence, Hermes thinks (head sways along), and the reply is spoken with audio-reactive motion.

4

It remembers

Turns chain into a named Hermes conversation, so the robot's memory persists across restarts — it's the same Hermes you use everywhere.

Features

Your agent, embodied Hermes' persistent memory, self-improving skills and tools — now with a face, a voice and antennas.
Hands-free, always on Adaptive voice activity detection with the mic array's speech flag; no buttons, no wake phrase to configure.
Expressive motion Antennas perk when it hears you, the head ponders while Hermes works, and speech drives natural head wobble.
Live settings page Status, mic meter, pause button, providers, voices and sensitivity — all adjustable from your browser while it runs.
Pluggable speech stack Speech-to-text via OpenAI or Groq; text-to-speech via OpenAI or ElevenLabs. One OpenAI key covers both defaults.
Check before you install The hermes-mini-check CLI verifies your Hermes server and API keys from any machine.
